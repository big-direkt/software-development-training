## 7. Exkurs: Markdown

- Warum es so wichtig und Quasi-Standard ist?
  - Lesbar im Rohformat + schön im gerenderten UI (GitHub README.md).
  - Standard für Projektbeschreibung, CONTRIBUTING, ISSUE Templates, Release Notes.
  - Unterstützt in allen Hosting-Plattformen.
- Grundlegende Markdown-Syntax
  - Cheat-Sheet: https://cheatsheets.zip/markdown

> [!TIP]
> Dieses Dokument, sowie die README und andere, sind in Markdown geschrieben, daher einfach den Code dazu anschauen.<br>
> Die Markdown-Syntax ist recht einfach und stark reduziert, doch genau dies macht es so einfach und daher auch weit verbreitet.
